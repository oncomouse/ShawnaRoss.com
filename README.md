# ShawnaRoss.com

## Directory Structure

```
source/
\-blog/ - blog controllers, partials, and feeds
\-images/ - images
\-javascripts/ - javascripts
\-layouts/ - layouts
\-posts/ - MD files storing blog posts
\-stylesheets - stylesheets
  \-fonts/ - Webfonts
  \-libraries/ - external SASS libraries
  \-settings/ - SASS configuration files
```
